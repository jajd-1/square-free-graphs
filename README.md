# Square-free graphs

Mantel's theorem states that the maximum edge count for a simple triangle-free graph with $n$ vertices is $\lfloor \frac{n^2}{4} \rfloor$, and that this maximum is obtained uniquely by the complete bipartite graph $K_{\lfloor\frac{n}{2}\rfloor, \lceil \frac{n}{2}\rceil}$.

The analogous problem for square-free graphs is less well-understood. Reiman derived the upper bound $$ex(n,C_4) \leq \frac{n}{4}(1+\sqrt{4n-3}),$$ which is asymptotically tight. Exact values for $ex(n,C_4)$ are known for $n$ up to roughly 40, and for most higher values of $n$ only lower bounds are known. The goal of this project is find improved lower bounds using evolutionary coding agents (and other AI/ML methods).
